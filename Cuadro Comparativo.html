Cuadro Comparativo: JQuery, AngularJS, React y AJAX(Jeremy Peña)

Tecnología Principales Comandos Descripción
JQuery
  • $(selector).action(): Selección y manipulación.
  • .hide() / .show(): Mostrar/ocultar elementos.
  • .ajax(): Manejo de solicitudes AJAX.
 Biblioteca de JavaScript para simplificar DOM, eventos y AJAX.
AngularJS
  • ng-model: Enlace de datos bidireccional.
  • ng-repeat: Iteración de datos.
  • $http: Solicitudes HTTP.
 Framework estructurado para crear aplicaciones web dinámicas.
React
  • React.createElement(): Crear elementos React.
  • useState(): Manejo del estado funcional.
  • useEffect(): Manejo de efectos secundarios.
 Biblioteca para construir interfaces de usuario basadas en componentes.
AJAX
  • XMLHttpRequest(): Solicitudes HTTP asincrónicas.
  • .open() / .send(): Configurar y enviar solicitudes.
  • fetch(): Manejo moderno de solicitudes HTTP.
 Técnica para cargar datos de servidores de manera asíncrona sin recargar la página.

Similitudes

  • JQuery, AngularJS, React y AJAX permiten trabajar con solicitudes HTTP para interacción con servidores.
  • Son compatibles con la mayoría de navegadores modernos.
  • Facilitan la creación de experiencias dinámicas e interactivas en la web.

Diferencias

  • JQuery es una biblioteca ligera; AngularJS y React son más complejos y orientados a SPA (aplicaciones de página única).
  • React utiliza un enfoque basado en componentes y AngularJS se basa en MVC (Modelo-Vista-Controlador).
  • AJAX no es una tecnología independiente, sino una técnica utilizada en combinación con otros.
